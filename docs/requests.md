# Request-Verarbeitung

## Externe Anforderungen

Cross-Repository-Anforderungen werden ausschließlich als External Tasks unter `external-tasks/open/` angenommen. Dateinamen und Frontmatter folgen ECO-ARC-0006.

## Interne Klassifizierung

Angenommene Anforderungen können zur Bearbeitung intern unter `incoming/` klassifiziert werden:

- `references/`
- `requested-foundations/`
- `review/`

Diese Ordner ersetzen weder den External Task noch dessen Statusführung. Die ursprüngliche Task-ID bleibt die verbindliche Referenz.

## Abschluss

Das Ergebnis wird im External Task dokumentiert. Danach wird dieser nach `external-tasks/done/` oder `external-tasks/rejected/` verschoben.
